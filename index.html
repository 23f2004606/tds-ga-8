# Simple linear forecast example
import numpy as np
from sklearn.linear_model import LinearRegression

# historical quarters (1..8) and revenue in $M
X = np.array([[1],[2],[3],[4],[5],[6],[7],[8]])
y = np.array([100,105,110,115,118,120,123,127])

model = LinearRegression().fit(X, y)
next_q = np.array([[9]])
print(f"Forecast revenue (Q9): ${model.predict(next_q)[0]:.2f}M")

Key Financial Formulas

Free Cash Flow (FCF):

\[\text{FCF} = \text{Operating Cash Flow} - \text{Capital Expenditures}\]

Discounted Cash Flow (DCF) — terminal value (Gordon Growth):

\[\text{TV} = \dfrac{FCF_{n+1}}{r - g}\]

Where: \(r\) = discount rate, \(g\) = perpetual growth rate.
