Sample: Simple Revenue Growth Calculation (Python)

def revenue_growth(prev, curr):
    """Return revenue growth rate as a percentage."""
    if prev == 0:
        return None
    return (curr - prev) / prev * 100

print(revenue_growth(120_000_000, 135_000_000))  # => 12.5

Financial Formula: Return on Equity (ROE)

We compute ROE using the standard formula:

\(\displaystyle ROE = \frac{Net\ Income}{Average\ Shareholders'\ Equity} \times 100\%\)

Example: If Net Income = \$50M and Average Equity = \$500M, then

\(\displaystyle ROE = \frac{50}{500} \times 100\% = 10\%\)
