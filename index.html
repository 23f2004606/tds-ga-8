Sample: Simple Revenue Growth Calculation (Python)

Function and example test cases (expected outputs shown in comments):

def revenue_growth(prev, curr):
    """Return revenue growth rate as a percentage (float rounded to 2 decimals).

    Returns None if previous value is zero (undefined growth).
    """
    if prev == 0:
        return None
    return round((curr - prev) / prev * 100, 2)

# Test cases
print(revenue_growth(120_000_000, 135_000_000))  # => 12.5
print(revenue_growth(100, 110))                  # => 10.0
print(revenue_growth(0, 50))                     # => None (undefined)

Financial Formula: Return on Equity (ROE)

We compute ROE using the standard formula:

\(\displaystyle ROE = \frac{\text{Net\ Income}}{\text{Average\ Shareholders'\ Equity}} \times 100\%\)

Example: If Net Income = $50M and Average Equity = $500M, then

\(\displaystyle ROE = \frac{50}{500} \times 100\% = 10\%\)
